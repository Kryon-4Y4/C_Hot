# 手机维修配件外贸数据管理系统

欢迎来到手机维修配件外贸数据管理系统的API文档站。

## 系统概述

本系统是一个完整的手机维修配件外贸出货数据管理平台，包含以下核心组件：

- **用户前台**: React自适应界面，支持数据查询和添加
- **API服务**: FastAPI提供RESTful接口
- **调度服务**: Celery异步任务调度
- **爬虫脚本**: UN Comtrade数据采集
- **数据库**: PostgreSQL数据持久化
- **运维后台**: 系统监控和运维操作
- **管理后台**: 账户和脚本管理

## 主要功能

### 数据管理
- 外贸出货数据查询和筛选
- 数据添加和编辑
- 数据确认流程
- 批量导入导出

### 爬虫调度
- 定时数据采集
- 多脚本管理
- 任务监控和日志
- 失败重试机制

### 统计分析
- 年度出口趋势
- 主要贸易伙伴分析
- HS编码分类统计
- 可视化图表展示

## 技术栈

| 组件 | 技术 |
|------|------|
| 前端 | React + TypeScript + Ant Design |
| 后端API | FastAPI + SQLAlchemy |
| 管理后台 | Flask + Flask-Admin |
| 数据库 | PostgreSQL |
| 缓存/队列 | Redis |
| 任务调度 | Celery |
| 容器化 | Docker + Docker Compose |
